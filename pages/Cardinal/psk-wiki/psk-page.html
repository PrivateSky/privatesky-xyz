This component is the most important component for documentation as it renders the entire page.

This component renders everything on UI starting with it's title property.

For loading it uses psk-ui-loader as a mean to mark the loading process. Psk-page listens to all the psk-send-chapter events sent by the psk-chapter.

For every event sent there are two possible situations that affects the hierarchy of chapters:

  • The hierarchy is not instantiated and therefore it will be created.
  • If it exists then it will be updated

If the loading has finished then the information between the two tags will be rendered on the UI in a child slot.
