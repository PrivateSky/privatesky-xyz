This component is extended from the psk-input component, it is used inside a psk-form component and is intending to be used as input for passwords.

The other components derived from psk-input are:

psk-text-input psk-email-input psk-date-input psk-number-input import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { password: { label: "Password", name: "password", required: true, placeholder: "Password here...", value: '' } } export default class FormComponentsExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); this.feedbackEmitter = null; this.on('openFeedback', (e) => { e.preventDefault(); e.stopImmediatePropagation(); this.feedbackEmitter = e.detail; }); let passwordSubmit = () => { this.feedbackEmitter(`We can't show the password you just entered, Thank you for your understanding!`,"number Example","alert-primary") } this.on("Password submit",passwordSubmit,true); } }

A very good example of using this component and the other form components can be found here: Using forms.
