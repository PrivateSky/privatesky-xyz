This component is extended from the psk-input component, it is used inside a psk-form component and is intending to be used as input for numbers as the component allows the user to enter only numbers.

The other components derived from psk-input are:

psk-password-input psk-text-input psk-date-input psk-email-input import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { number: { label: "Choose a number:", name: "number", required: true, value: '0' } } export default class FormComponentsExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); } }

A very good example of using this component and the other form components can be found here: Using forms.
