This component is part of the form components and is inteded to display a text inside the form, or a label for a form component (check Using Forms for details).

The components, by their logic, already have the possiblity to display a label for them, but there are always different needs.

So this component is ment to please your needs to display a label anywhere you want it.

A text representing the label to be displayed in case the attribute "label" is not provided.

 import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { name: { label: "Name", name: "name", required: true, placeholder: "Name here...", value: '' }, age: { label: "Age", name: "age", required: true, placeholder:"Enter you age here...", value:'' }, adress: { label: "Adress", name: "adress", required: true, placeholder:"Enter you adress here...", value:'' } } export default class FormComponentsExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); } } Enter your age Enter your adress Enter your age Enter your adress
