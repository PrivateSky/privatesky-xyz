Psk-Radio component is designed to be used inside a psk-radio-group component because in the essence, only one radio can be selected inside a group.

If you want to be able to select multiple options from a list, you can use Psk-Checkboxes or Psk-Select

Radios are typically rendered as small circles, which are filled or highlighted when selected.

Psk-radio-group is notified when a psk-radio is clicked, so the group can then update the model.

 gender: { label: "Select your gender", required: true, options: [{ label: "Male" }, { label: "Female", value: "F" } ] }

An explanatory example using a full form can be found at: Using forms page.
