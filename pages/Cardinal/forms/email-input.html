This component is extended from the psk-input component, it is used inside a psk-form component and is intending to be used as input for emails as it internally validates the email format.

The other components derived from psk-input are:

psk-password-input psk-text-input psk-date-input psk-number-input import ContainerController from "../../cardinal/controllers/base-controllers/ContainerController.js"; const model = { email: { label: "Email ", name: "email", required: true, placeholder: "Email address here...", value: '' } } export default class FormComponentsExampleController extends ContainerController { constructor(element) { super(element); this.model = this.setModel(JSON.parse(JSON.stringify(model))); } }

A very good example of using this component and the other form components can be found here: Using forms.
