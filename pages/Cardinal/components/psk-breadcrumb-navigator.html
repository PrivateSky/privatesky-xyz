The purpose of the component is to display the navigation tree of the user (like a file explorer) with the possibility to navigate trough the segments.

Becaue of flexibility reasons, the navigation needs to be handled inside the controller.

At this moment, the component accepts two attributes: event-name and segments.

Event name is optional and is used to trigger an event when the user is clicking o the breadcrumb segments. Segments is an array of Objects wth the following structure:

{ label: string // The label to be displayed path: string // The path for navigation. Ths attribute is sent as event-data with psk-button component. }

The component accepts a special attribute, mode, which allows the user to choose between three default styles of breadcrumb:

default: The default mode of the component. Also this is the selected mode if none provided. scroll: By using this mode, the navigator will display a scrollbar on X-axis of the component for an easier navigation over the clickable segments. arrows: By using this mode, the component will display two arrows, one on the left side and one on the right side for an easier navigation over the clickable segments.

The compnent accepts no HTML children.
