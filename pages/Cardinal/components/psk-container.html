This component uses the controller passed in controllerName(inside the ControllerFactory service) or the default Controller.

It can also render javascript code inside the component if it is written between the script tags:

Inside a psk-container component any other psk components can be used,even the psk-container itself.

In case there is an already created component , and the user wants to provide the HTML context to the container, we implemented a feature that will begin the searching of a controller script from there: parentHost.

It also uses psk-page-loader only if the htmlFilePath property is provided.

This component has a slot tag so the data you enter between the tags can be rendered before the content obtained from the htmlFilePath page (if the property is given).
