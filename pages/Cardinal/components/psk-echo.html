This is one of the simplest components of CardinalJS. Its role is to render values ​​that have been dynamically set in the page controller.

The psk-echo component must be used in close connection with the psk-container.
To use this component at full capacity, read also about psk-for-each.

In order to generate content in a simple form, without inputs you can use this component.

This component has no HTML children.
