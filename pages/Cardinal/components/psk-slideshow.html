This components purpose is to create a slideshow of images using a given images source string: images.

In order to create the nice opacity switching effect between the images we use a function where we switch between an opacity of 1 or 0 this way:

At the two end, 0% and 100%, and when the animation time expires(at the moment we use 10), then the opacity is considered 0 and in any other moment of the animation the opacity is 1.

Because some images may not be the same size we made a fucnction that will resize the height of the slideshow everytime it is necessary.

If the title,caption or if the element has any HTML children, then an auxiliary div container is created with content class and all given info in the three at question.

If there are any actions (any children) then a slot will be created (inside a actions classed div) where any info will be rendered without any modifications made.
