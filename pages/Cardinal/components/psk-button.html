Basic button component used to trigger button events for any need.

This button is not like standard HTML button, which in HTML it submits a form, instead it is able to trigger custom button events with custom data to be handled in a parent controller.

The button comes with a default style from boostrap using primary color(blue), but it can be easily designed using CSS files according to the selected theme for the web-site.

Recommended to be used in case of displaying a label or an icon. To be used if attribute label is not defined.
