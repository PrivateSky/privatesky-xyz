This component role is to create a link that will respond with a redirect inside that website the moment it is clicked.

The path to redirect is given using the properties and if it is not given then the component will not be rendered.

 The use of tag attributes is encouraged because the psk-link will support menu page changes in app-config.json in this way. If you opt out for page attribute, you will have to manage by yourself the url changes.

This component child is a span that will render the info on an newly created button.
