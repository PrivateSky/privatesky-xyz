The setup process of an development environment for building and running SSApps. For example, we use tutorial-workspace. First of all, create a local clone of the tutorial-workspace Github project: git clone https://github.com/PrivateSky/tutorial-workspace.git Enter into checkout folder and run the following command: npm run dev-install Run the brick storage and static file http server npm run server Wait for a message like "Domain boot successfully" and then open a new terminal and build all the ssapps, wallets, etc in the current project npm run build-all Finally, open the browser ( Chrome in incognito is the best) and go to http://localhost:8080/tutorial/loader The setup process of PrivateSky (contributors project version) can be reduced to git clone the project and run npm install process. (steps bellow) First of all, create a local clone of the PrivateSky Github project: git clone https://github.com/PrivateSky/privatesky.git Enter into checkout folder and run the following command: cd privatesky && npm install
npm install will install octopus that will handle and clone all modules from git, similar behaiviour as in a project with submodules. First of all, create a local clone of the PrivateSky Github project: git clone https://github.com/PrivateSky/psk-release.git Enter into checkout folder and run the following command: cd privatesky npm install The Node version is checked and if is all good the installation process begins by downloading the required dependencies. Once the installation process has finished successfully, the following command needs to be run: npm run start At this point, the PrivateSky installation is done and the PrivateSky Node is up and ready to go! To update an existing PrivateSky installation run command: npm run update
